# Packet Analysis Tool

## Overview

This project demonstrates packet capture and network traffic analysis using Wireshark, tcpdump, and Scapy. The objective is to monitor network communication, inspect network protocols, and understand how data flows across a network.

## Objectives

* Capture live network traffic
* Analyze DNS, TCP, UDP, ICMP, and NTP packets
* Monitor network communication in real time
* Perform packet inspection using Wireshark
* Understand packet structures and protocol behavior

## Tools Used

* Wireshark
* tcpdump
* Scapy
* Python
* Kali Linux

## Project Activities

* Captured live network packets
* Analyzed DNS queries and responses
* Inspected ICMP packets generated by ping commands
* Examined TCP communication and packet details
* Observed ARP and NTP traffic
* Performed packet sniffing using Scapy

## Skills Gained

* Network Traffic Analysis
* Packet Inspection
* Protocol Analysis
* Network Monitoring
* Cybersecurity Fundamentals
* Wireshark Filtering Techniques

## Results

* Successfully captured and analyzed live network traffic
* Identified DNS, TCP, UDP, ICMP, ARP, and NTP packets
* Improved understanding of network communication and packet structures
* Gained hands-on experience with industry-standard packet analysis tools

## Conclusion

This project provided practical experience in packet analysis and network monitoring using Wireshark, tcpdump, and Scapy. It helped build foundational cybersecurity skills that are essential for SOC Analyst, Network Security, and Penetration Testing roles.
